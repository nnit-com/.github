## Hi there 👋
chatbot

1. Objective

本文描述了chatbot（1.0）的详细技术设计。chatbot是企业内部使用的智能聊天机器人，为用户提供7*24小时自助问答服务。用户可通过企业微信以及Teams聊天工具，直接与chatbot对话，获取相关的问题的答案以及有用的信息，同时还可以使用chatbot实现自助预订Bizconf视频会议以及工作时间转人工支持服务。
